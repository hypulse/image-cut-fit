# image-cut-fit

AI 생성 이미지나 에셋을 활용해 빠르게 게임이나 앱 데모를 만들고 싶어서 개발한 로컬 앱입니다.
배경과 빈 여백을 정리해 PNG 스프라이트로 저장하고, 한 장에 모인 스프라이트도 개별 이미지로 분리할 수 있습니다.

## 실행

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

`rembg`는 첫 실행 시 모델 파일을 다운로드할 수 있습니다.

## 사용 방법

1. `이미지 업로드`에서 PNG, JPG, JPEG, WebP 파일을 추가합니다.
2. 한 장에 여러 객체가 있다면 `스프라이트 시트 분리`를 켜고 `최소 영역`으로 작은 조각 포함 여부를 조정합니다.
3. 각 이미지 카드에서 `Original`, `Cropped`, `Rotated`, `Output`을 비교하며 배경 제거와 출력 크기를 맞춥니다.
4. 배경 제거가 어색하면 모델, `Alpha`, `내부 복원`, `마스크`를 조정합니다.
5. `가로`, `세로`, 비율 잠금, `리사이즈`로 최종 크기와 배치 방식을 정합니다.
6. 필요하면 `수동 지우기`를 켜고 브러시로 추가 제거 영역을 그립니다.
7. `각도`로 이미지를 회전하고 상/우/하/좌 `Padding`을 추가합니다.
8. 불투명 배경이 필요할 때 `투명 배경`을 끈 뒤 `배경색`을 선택합니다.
9. 개별 결과는 `PNG 다운로드`나 `저장`으로, 전체 결과는 `모든 이미지 저장`으로 `outputs/`에 저장합니다.

## 옵션 가이드

- `u2net`은 기본값, `isnet-general-use`는 일반 이미지 보존, `isnet-anime`은 일러스트/애니풍 이미지에 적합합니다.
- `모델 선택하지 않음`은 배경 제거를 건너뛰고 원본 alpha 기준으로 crop/분리합니다.
- `Alpha`를 높이면 흐린 잔여 배경을 더 강하게 잘라내지만, 얇은 디테일이 사라질 수 있습니다.
- 모델 결과에서 남은 객체를 찾지 못하면 원본 기준으로 편집을 계속할 수 있습니다.
- `수동 지우기`는 크롭된 이미지에 적용되며, 그린 영역을 투명하게 만든 뒤 다시 자동 crop합니다.
- `각도`는 크롭된 이미지에 적용되며, 양수는 시계 방향으로 회전합니다.
- `Padding`은 상/우/하/좌를 각각 따로 설정할 수 있습니다.
- `비율 유지 중앙`은 형태를 유지해 투명 캔버스 중앙에 놓고, `늘려서 채우기`는 지정한 크기에 맞춰 비율을 무시하고 늘립니다.
- AI로 원본을 만들 때는 단색 배경과 충분한 객체 간격을 요청하면 분리 결과가 안정적입니다.
